# Sales Forecasting: Regression Models Benchmark

This project demonstrates a complete end-to-end Machine Learning pipeline for predicting sales data. It compares the performance of classical linear models against advanced ensemble techniques.

## 🚀 Models Implemented:
- **Linear Models**: Linear Regression (from scratch), Ridge, and LASSO.
- **Support Vector Machines**: SVR (Support Vector Regressor).
- **Tree-Based Models**: Decision Tree & Random Forest.
- **Advanced Boosting**: LightGBM & CatBoost.

## 🛠️ Skills Demonstrated:
- **Data Preprocessing**: Handling tabular data, scaling, and feature engineering.
- **Model Evaluation**: Comparing models using metrics like RMSE, MAE, and R-squared.
- **Frameworks**: Scikit-Learn, LightGBM, CatBoost, and NumPy for scratch implementation.

## 📊 Objective:
To identify the most robust model for capturing trends in tabular sales datasets and understanding the impact of regularization (L1/L2) on model generalization.
